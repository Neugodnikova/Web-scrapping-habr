# Web Scrapping Habr

Этот проект предназначен для парсинга статей с сайта [Habr](https://habr.com/ru/articles/) с целью поиска статей по ключевым словам в заголовках и содержимом.

## Описание

Скрипт анализирует страницы со свежими статьями на Хабре, извлекая информацию о статьях и проверяя их на наличие ключевых слов в заголовках и тексте. Статьи, которые содержат ключевые слова, выводятся в консоль.

## Функциональность

- Скрипт собирает список свежих статей с Хабра.
- Ищет ключевые слова в заголовках статей.
- Дополнительно, проверяет текст статьи целиком на наличие ключевых слов.
- Выводит два списка:
  - Статьи, найденные по анализу **превью** (аннотации).
  - Статьи, найденные по анализу **полного текста** статьи.
